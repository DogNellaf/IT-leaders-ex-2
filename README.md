# KNN и Линейная регрессия на данных Google Play Store

Проект демонстрирует применение алгоритмов машинного обучения (KNN и линейная регрессия) для анализа данных мобильных приложений из Google Play Store.

## Описание проекта

Этот проект содержит практическую реализацию двух популярных алгоритмов машинного обучения:
- **KNN (K-Nearest Neighbors, k-ближайших соседей)** — алгоритм, основанный на поиске k ближайших соседей для классификации и регрессии
- **Линейная регрессия** — метод предсказания непрерывных значений на основе линейной зависимости

Данные для анализа взяты из Google Play Store и содержат информацию о приложениях, включая рейтинги, количество установок, категории и другие характеристики.

## Датасет (Google-Playstore.csv)

Файл содержит информацию о приложениях с следующими столбцами:
- **App Name** — название приложения
- **App Id** — уникальный идентификатор
- **Category** — категория приложения
- **Rating** — рейтинг приложения (0-5)
- **Rating Count** — количество оценок
- **Installs** — количество установок
- **Price** — цена приложения
- **Size** — размер приложения
- **Minimum Android** — минимальная версия Android
- **Developer Id** — идентификатор разработчика
- **Released** — дата выпуска
- **Last Updated** — дата последнего обновления
- **Content Rating** — возрастной рейтинг контента
- **Ad Supported** — поддержка рекламы
- **In App Purchases** — наличие покупок в приложении и другие параметры

## Требуемые библиотеки

Проект использует следующие Python библиотеки:
- **numpy** — численные вычисления
- **pandas** — обработка и анализ данных
- **matplotlib** — визуализация данных
- **scikit-learn** — инструменты машинного обучения

### Установка зависимостей

Все необходимые пакеты устанавливаются автоматически при запуске ячеек notebook'а. Также можно установить их вручную:

```bash
pip install numpy pandas matplotlib scikit-learn
```
